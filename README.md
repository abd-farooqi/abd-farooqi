## 📂 My Standard Scalable Directory Layout

I enforce strict, self-documenting directory paths in my repositories to ensure clean file management:

```text
src/
├── components/
│   ├── ui/           # Radix & shadcn primitive primitives (Atoms)
│   ├── modules/      # Composite feature layouts (Molecules)
│   └── compounds/    # Complex domain layouts (Compound patterns)
├── store/            # Lightweight Zustand global state slices
├── hooks/            # Custom reusable React hooks
├── app/              # Next.js App Router structural pages
└── utils/            # Strictly typed helper functions
```
